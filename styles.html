Style Examples

All styling follows Flat design, a minimalistic UI design genre that is adopted by all major mobile OS and works well cross platforms.

Specifically, we abandon the use of shadows and depth to avoid multi-level complex UI. The only exceptions are the main navbar and the default dropdown list. Please don't use shadow in your design if you don't want to go through prolonged design review.

We also dilute the definition of the border of UI containers and eliminate any unnecessary decoration, including getting rid of seams in the list and only apply seams when you can categorize the items inside the list. By doing so, we create a very subtle styling to allow the content to take the absolute dominant position in the visual hierarchy.

The ultimate goal of this UI styling is to create a clean and immersive environment for consuming content.
